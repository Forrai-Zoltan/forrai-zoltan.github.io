Forrai Zoltán ~

Hi there! My name is Forrai Zoltán,1 I'm a writer & scholar living in Budapest, Hungary.

I'm interested in many different things, from the humanities to technology. I have what you might call 'can't sit on his ass syndrome' — it's terminal. I constantly require something to tinker with, or else I go insane.

This website was partially built in an attempt to break away from social media, AI slop and web 2.0/3.0 in general. I handwrite the code myself using HTML and CSS. This website is not yet mobile friendly.

  1. We use the eastern naming order. Forrai is my family name, Zoltán is my given name.
